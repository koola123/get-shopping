# Technology stack:
- React
- React useContext API
- React hooks
- Tailwind CSS
- Parcel
- Fully mobile responsive

## Link to API:
This project makes use of the Fakestore API for receiving the product data.
``` 
https://fakestoreapi.com/ 
```
